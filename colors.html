Colors in HTML

RGB model

  • Every color can bve represented by a combination of RED,GREEN, and BLUE
  • Each of these 3 base colors can take a value between0 and 255 which leads to 16.8 million different colors

Defning colors in CSS

  • RGB/ RGBA Notation

    Regular RGB Model

    1. rgb(0,255,255)
    2. rgb(244,179,63)

    RGB with transparency("alpha")

    1. rgba(0,255,255,0.3)
    2. rgba(244,179,63,0.7)

  • Hexadecimal Notation

    Instead of using a scale from 0 to 255, we go from 0 to ff( 255 in hexadecimal numbers)

    1. #00ffff
    2. #0ff [Shorthand]

In practice, we mostly use hexadecimal colors, and rgba when we need transparency
